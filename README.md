# Lock In - AI Personalized Study Tracker
The AI Personalized Study Tracker is a web application designed to help students create customized study plans. Using AI, the system dynamically adjusts schedules based on difficulty levels, available time, upcoming exams, and completed or missed tasks. The goal is to improve learning efficiency, reduce stress, and help students stay consistent with their studies.

>[!NOTE]
> This mobile application is built using Replit AI primarily for testing and rapid prototyping. <br>
> It demonstrates the core functionality of a personalized AI-powered study tracker.<br>
> including dynamic plan generation, progress tracking, and dashboard visualization. <br>
> The current implementation is optimized for experimentation and iterative development.<br>
> and can be extended or deployed on other platforms for production use.

### Real-World Problem in EdTech
Many students struggle with managing their study time effectively. Standard study schedules often fail to account for:
- Different learning speeds for different subjects.
- Changing priorities due to upcoming exams or assignments.
- Inconsistent study habits and missed tasks.
- Lack of personalized guidance to optimize productivity.

This leads to stress, wasted study time, and poor academic performance.

### How This Project Solves the Problem
The AI Personalized Study Tracker addresses these challenges by:
- Creating a study plan tailored to each student's needs and available time.
- Dynamically updating the schedule when tasks are completed or missed.
- Prioritizing difficult topics and upcoming deadlines.
- Providing a dashboard to track progress and maintain consistency.
- Offering a complete end-to-end solution from planning to progress monitoring.
